# Annu Jindal Portfolio

Premium cinematic portfolio built with Next.js for Vercel deployment.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS v4
- Vercel Analytics
- Vercel Speed Insights

## Development

```bash
npm run dev
```

## Deployment

This project is structured for Vercel out of the box. Import the repository into Vercel and deploy the default branch you want to publish.

## Content updates

- Main content lives in `src/data/portfolio.ts`
- Page sections live in `src/components/home`
- Final portfolio images can be added under `public/images`
